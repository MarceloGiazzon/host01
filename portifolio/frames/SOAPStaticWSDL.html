SOAP Static WSDL

Category: Genexus External Objects

This Java Servlet Filter intercepts incoming HTTP requests, specifically looking for the presence of the ?wsdl query string, which indicates a request for a Web Service Definition Language file. When a WSDL request is detected, the filter stops the normal servlet processing chain and instead serves a static WSDL file (e.g., ws_efactura_intercambio.wsdl) located in a predetermined path.
